# PALM Device Overview
Corporate Engineering Summary

## Purpose
The PALM Device is designed to support the complete Blue Vizion Identity Intake Workflow, integrating multiple sensing, scanning, and verification modules into a single physical unit.

## Key Functions
- Voice intake (citizenship translator)
- Document scanning
- Biometric capture (palm and thumb)
- Identity verification
- Document authentication
- Data transfer to Blue Vizion Worker API
- Lineage & estate verification support
- Ledger-bound output packaging

## Target Deployment
- Intake offices
- Remote-field verification stations
- Estate-lineage adjudication centers
- Mobile identity teams
