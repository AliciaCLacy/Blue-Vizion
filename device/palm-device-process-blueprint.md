# PALM Device — Process Integration Blueprint

This document explains how the PALM device integrates into the Blue Vizion 20-step Intake Workflow.

## Blueprint Reference
**Image:** `images/palm-device-blueprint.png`

The blueprint outlines:

### 1. Octagonal Frame
Structural housing for all primary modules.

### 2. Document Verification Rails
Two angled rails guiding physical documents across the scanning sensor.

### 3. Biometric Thumb Module
Capacitive or optical sensor for thumb authentication.

### 4. Palm Seal Surface
Top-surface palm biometric reading area (vein + print).

### 5. Power and Data Ports
Rear panel ports for power input & data transmission (USB-C / PoE).

### 6. Directional Intake Flow
Arrows indicate user-guided workflow:
- Document → biometric → confirmation → ledger packet.
