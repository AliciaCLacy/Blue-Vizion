## Overview

The PALM Device is a multi-module biometric intake unit designed to perform identity, document, and biometric verification for Blue Vizion's unified intake workflow.

It supports:

- Citizenship interpretation (audio → structured data)
- Document capture and verification
- Personal questions intake
- Biometric palm and thumb authentication
- Estate, lineage, and descendant checks
- Secure ledger-bound output
