# PALM Device — 3D Concept Rendering Description

This document describes the industrial 3D rendering of the PALM Intake Device.

## Render Reference
**Image:** `images/palm-device-3d-concept.png`

## Design Language
- Monolithic octagonal body
- Sandstone matte finish
- Chamfered document rail with recessed scanner
- Floating palm sensor glass disc
- Illuminated PALM emblem ring
- Rear metal port panel

## Color Palette
- Exterior Shell: Deep Cerulean Blue (#142A44)
- Sensor Housing: Matte Sand (#F6E8D0)
- LED Ring: Soft Neon White (#E8F3FF)

## Ergonomics
- 12° document rail angle
- 3-point stabilizing base
- Thumb sensor located left for single-handed operation

## Production Notes
This concept is manufacturing-ready but does not include internal PCB layout.
